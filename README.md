IPL Cricket Analysis and Prediction System
Project Overview

This project analyzes historical IPL (Indian Premier League) match data to understand team performance, player statistics, and match trends. Data science techniques such as data preprocessing, exploratory data analysis (EDA), and visualization are used to uncover insights from the dataset.

Dataset Description

The project uses two datasets:

matches.csv
Contains match-level information such as season, teams, toss details, match winner, venue, and umpires.

deliveries.csv
Contains ball-by-ball match data including batting team, bowling team, batsman, bowler, runs scored, and dismissal type.

Data Preprocessing

The following preprocessing steps were performed:

Removed duplicate values

Handled missing values

Merged matches and deliveries datasets

Created new features such as total runs and match statistics

Exploratory Data Analysis

The project includes analysis of:

Team performance across seasons

Player statistics and comparisons

High scoring matches (200+ runs)

Runs scored per over

Score distribution by innings

Orange Cap winners by season

Data Visualization

Various charts were created to present insights:

Bar charts

Line graphs

Heatmaps

Box plots

Scatter plots

Key Insights

Some teams perform consistently well across seasons.

Certain players score more runs consistently.

High scoring matches are common in recent IPL seasons.

Teams have different scoring patterns across overs.
